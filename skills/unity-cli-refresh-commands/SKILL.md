---
name: unity-cli-refresh-commands
description: >
  Refresh cached custom command list from Unity
---

Refresh the cached list of user-defined custom commands from the running Unity Editor.

Steps:

1. Run:
```bash
python "${CODEX_PLUGIN_ROOT}/cli/cs.py" list-commands --json --project "$(pwd)"
```

2. Parse the JSON output. Extract the `data.commands` array.

3. Filter out built-in commands. The following namespace/action pairs are built-in and must be excluded:
   - editor/*
   - gameobject/*
   - component/*
   - transform/*
   - material/*
   - prefab/*
   - project/*
   - scene/*
   - screenshot/*
   - profiler/*
   - session/*
   - command/*

4. If no custom commands remain after filtering, report "No custom commands found" and delete `${CODEX_PLUGIN_ROOT}/skills/unity-cli-command/dynamic-commands.md` if it exists.

5. If custom commands exist, write them to `${CODEX_PLUGIN_ROOT}/skills/unity-cli-command/dynamic-commands.md` in this format:

```markdown
# Custom Commands (auto-generated)

> Generated by $unity-cli-refresh-commands. Do not edit manually.
> Re-run $unity-cli-refresh-commands after registering new C# commands.

| namespace | action | summary | args |
|-----------|--------|---------|------|
| ... | ... | ... | ... |
```

6. Report how many custom commands were cached.
